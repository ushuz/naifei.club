# Spotify苹果最新版下载

Spotify iPhone/iPad：在中国国区的app store搜索不到Spotify App，只能切换外区AppleID来下载Spotify 独享美区AppleID ，为了防止出现锁机问题，请严格按照下面流程处理，如果导致锁机，我们提供不了任何解决方案！

iOS下载spotify的流程，iPhone、iPad下载Spotify也能用：

  1. 打开App store,点击右上角头像,滑到底部
  2. 点击【退出登陆】
  3. 获取美区AppleID，独享美区AppleID
  4. 输入ID，点击登陆，选择【其他选项】->【不升级】
  5. 下载Spotify App，退出美区AppleID

下面是 iOS下载spotify图文教程，iPhone、iPad下载Spotify通用：

1、进入app store，点击右上角的头像，进入账户页面，然后一直下滑到最底部。

2、看到退出登录（sign out）后点击，返回最上面，看到ID变为空。

3、为了避免Spotify苹果下载app的 AppleID被滥用，就不直接放在这里了，需要的朋友直接购买即可：美区AppleID ，填写“数字严选”提供的AppleID和密码并登录。

4、iPhone/iPad用户填写上面的Apple ID和密码，按照下面的图片顺序，首先点击账号下面的“登录”或者sign in，如果出现图二页面，点击底部的“其他选项”，弹出第三张图片，选择“不升级”，没有其他问题的话，就成功了，点击app store右下角的放大镜图标，搜索“Spotify”直接下载就好了。

5、!!!!!!重点：为了方便其他人使用，下载Spotify APP完成后请退出外区账户，和第二步一样，滑到账户的最下面，看到退出登录或者sign out后点击，返回最上面，看到Apple ID变为空即可。

下载Spotify APP常见问题：如果Apple ID登陆不上，提示账号密码错误或没反应或忘记密码等，检查是否填写错误。或者全部重新输入。实在不行等待15分钟重试。如果依然不行或者提示ID被锁定，请留言讨论。
