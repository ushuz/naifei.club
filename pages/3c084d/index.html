# Spotify

Spotify，又称思播，声田，声破天。是一家在线音乐流服务平台。目前是全球最大的流音乐服务商之一。和全球四大唱片公司：环球音乐、索尼音乐娱乐、华纳音乐、腾讯音乐娱乐以及其他唱片公司都有合作授权，截止到2021年初，活跃用户规模大约3.5亿元，付费用户规模1.5亿。

2018年4月3日，Spotify在纽交所上市，当天市值约265亿美金。
Spotify的营收来自两个方面。免费服务的收入主要来自于广告。付费服务的收入来自Spotify Premium/会员用户每月支付的费用。

# Spotify免费版对比Spotify会员

Spotify提供免费增值服务模式，有两种流音乐：Spotify Free（160kbit/s）和Spotify Premium（320kbit/s以上）。Spotify支付按照歌曲中的点击率和播放次数比例，每次播放大约为0.04-0.1 美分。其与苹果公司的iTunes是竞争关系，苹果后推出Apple Music与其竞争。

# Spotify免费版

Spotify为免费用户提供七种不同类型的广告，Spotify免费版的广告包含音乐广告、显示广告、广告商广告、主页广告、品牌化播放清单、灯箱和广告商的网页，这些广告的尺寸，类型和用户参与度有所不同。2009年2月，非付费用户的广告为15秒。2020年5月，Neowin网站报导这已经上升到约30秒。
