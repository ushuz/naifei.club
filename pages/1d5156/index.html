# YouTube是什么

YouTube又称油管。YouTube 是世界上最大的视频搜索和分享平台，YouTube的用户几乎遍布全球。与之相对应的，每一个用户几乎都可以在YouTube上看到他想看的任何视频内容。这也是众多用户对YouTube趋之若鹜的原因。

2018年11月,YouTube正式推出 YouTube Premium （又称YouTube会员 ）和YouTube Music会员 服务。

# YouTube怎么读

YouTube读法：英['ju:tju:b]，YouTube 中文可以读作：油管

# youtube的功能

YouTube作为当前行业内在线视频服务提供商，YouTube的系统每天要处理上千万个视频片段，为全球成千上万的用户提供高水平的视频上传、分发、展示、浏览服务

# 影片播放

YouTube视频可在一个范围内选择所想要的像素水平，分别以标准质量（SQ），高品质（HQ）和高清晰度（HD）来取代之前以数值代表垂直分辨率的方法。而当前以H.264/MPEG-4 AVC作为缺省的视频编解码器、音频则改为AAC格式

# 影片上传

YouTube的影片上传支持大多数常见的视频文件格式，包括AVI、MKV、MOV、MP4、DIVX、FLV和THEORA、MPEG-4、MPEG和WMV等格式。此外也支持3GP，让上传者也可使用移动电话来上传影片。

# 电影频道

YouTube的电影频道已经上线许久，里面大部分内容需要付费，其价格从1.99美元起，用户购买的是48小时观看权。里面也有部分免费电影观看

# 视频直播

YouTube Live于2012年发布，为免费的基于广告的一向业务。YouTube一直都在考虑如何让YouTube Live平台不再只是一项免费的、基于广告的服务，而该公司正在推行这项计划。

# YouTube music（YouTube音乐）

YouTube Music是一款新的音乐流媒体服务，该新服务将包括一款重新设计的手机应用以及一款新的电脑端音乐播放器。2018年5月22日，谷歌旗下视频网站YouTube将推出音乐流媒体服务YouTube Music。
